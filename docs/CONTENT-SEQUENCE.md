# Content Sequence

## Working Rule

后续内容补全统一按：

- 时间线
- 专辑
- 歌曲

这样做的原因：

- 先确定时期，能避免歌曲解读彼此脱节
- 先完成专辑层理解，歌曲页会更有整体脉络
- 逐首写歌时，更容易保持风格、主题和时代感一致

## Suggested Editing Passes

### Pass 1

- 确认时间线节点描述
- 确认专辑气质与代表作
- 每首歌先完成整体解读

### Pass 2

- 完整歌词录入
- 逐段 / 逐句细读
- 关键信息补齐：MV、标签、关联歌

### Pass 3

- 整首歌设计分析
- 专辑内部歌曲间关系
- 首页推荐与情绪推荐的说明文字优化
